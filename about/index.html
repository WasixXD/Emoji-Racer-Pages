About the race

Emoji Grand Prix is a social betting experiment powered by short-form video comments. Each generated race has a set of emoji competitors, and TikTok viewers can place a bet by commenting in the format bet 30 🐴.

Comments from the latest active TikTok race create bettor accounts when needed, keep each user's highest bid, and publish this static dashboard from the latest race data. The goal is to make the race feel alive even outside the original video feed: viewers can see which emoji is attracting the most money, what the current odds are, and what every bettor stands to win.
